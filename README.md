# homebrew-tap

Homebrew formulae for [eharriett0](https://github.com/eharriett0) tools.

## Install

```sh
brew install eharriett0/tap/wt
```

or

```sh
brew tap eharriett0/tap
brew install wt
```

## Formulae

| Formula | Description |
|---|---|
| [`wt`](https://github.com/eharriett0/wt) | Multi-window git coordination: per-window worktrees, claim/release work, file-level collision detection across windows, merge/push guards. |
